# 解放台湾 - 塔防

消灭绿娃，解放台湾全境。

## 游戏简介

这是一款纯前端塔防游戏，以解放台湾为背景，玩家部署军事炮塔阻击绿色青蛙敌人（代号"绿娃"），从南到北逐步解放台湾西部各大城市。

## 特色

- **青蛙敌人**：5种绿蛙敌人——步兵蛙、侦察蛙、装甲蟾蜍、医疗蛙、蛙王Boss
- **军事炮塔**：155mm榴弹炮、岸防速射炮、红旗-9导弹、电磁轨道炮
- **拖拽部署**：点击塔卡片后鼠标跟随预览，点击地图部署
- **自动升级**：炮塔击杀敌人自动升级，Lv1→Lv2需8杀，Lv2→Lv3需20杀
- **五大战役**：高雄港→台南城→台中防线→新竹基地→台北决战
- **连击系统**：连续击杀触发金币倍率加成
- **粒子特效**：爆炸、烟雾、升级光环等粒子效果
- **屏幕震动**：Boss死亡时触发震屏效果
- **存档系统**：支持localStorage存档/读档

## 快速开始

直接用浏览器打开 `index.html` 即可运行，无需服务器、无需安装。

```
双击 index.html
→ 点击「开始作战」
→ 选择炮塔拖拽到草地上
→ 阻击绿娃，解放台湾！
```

## 操作说明

| 操作 | 说明 |
|------|------|
| 点击塔卡片 | 选中炮塔，鼠标跟随预览 |
| 点击草地 | 部署炮塔（每次部署后自动取消选中） |
| 右键草地 | 取消当前选中 / 升级已有炮塔 |
| 右键已部署炮塔 | 手动升级（需消耗金币） |
| 快捷键 1-4 | 快速选择炮塔 |

- 纯 HTML + CSS + JavaScript
